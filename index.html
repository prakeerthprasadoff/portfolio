Global ML + analytics system

Global AI Adoption Prediction System

Built an end-to-end machine learning system to analyze and predict global AI adoption using multilingual survey data across 21 countries, revealing behavioral and demographic adoption drivers.

  • Model: HistGradientBoostingRegressor with Scikit-learn preprocessing pipeline
  • NLP stack: langdetect + translation APIs for 12-language normalization at 23,000+ response scale
  • Data tooling: OpenRefine + feature engineering to build an AI Adoption Score across use cases
  • Analytics: Tableau dashboards for trust, usage behavior, and demographic driver insights
Stack

Python, Scikit-learn, OpenRefine, Tableau, NLP tooling, feature engineering

Impact

Identified trust as the strongest adoption driver and informed targeting strategies that improved conversion estimates by ~62%.

Deep learning + video understanding

MLB Pitch Classification System

Built a full video classification pipeline that combines spatial and temporal modeling to identify pitch types from baseball video sequences.

  • Core model: ResNet-18 + LSTM hybrid for spatial-temporal pitch classification
  • CV pipeline: OpenCV frame extraction with PyTorch sequence training workflow
  • Benchmarks: Decision Tree + Random Forest baselines using Scikit-learn
  • Data strategy: Class balancing from ~100 to ~200 samples per pitch type
Stack

Python, PyTorch, OpenCV, ResNet, LSTM, Scikit-learn

Impact

Improved classification accuracy from ~56% to ~74%, validating CNN + sequence modeling for video tasks.

Course Prerequisite Recommender (KRR)

Built an intelligent planner that combines graph-based prerequisite modeling with LLM reasoning to suggest alternative graduation pathways.

  • Symbolic core: Directed prerequisite graph modeled in Python with NetworkX
  • Algorithms: Graph traversal for dependency chains, blockers, and missing requirements
  • LLM layer: Azure OpenAI for alternative path generation and requirement completion ideas
  • UX: React planner for interactive path exploration and course recommendation review
Stack

React, Python, NetworkX, Azure OpenAI APIs

Impact

Enabled students to evaluate optimal and flexible graduation routes with hybrid symbolic + LLM reasoning.

Smart Vision Glasses for Navigation

Developing a real-time assistive navigation system for visually impaired users by combining smart glasses, sensor fusion, and computer vision.

  • Sensor fusion: TFMini LiDAR + BNO085 IMU via Arduino for distance and orientation signals
  • Vision stack: OpenCV + Mediapipe pipeline for obstacle and scene interpretation
  • Connectivity: BLE/WiFi packet analysis to reverse-engineer HeyCyan data transfer flow
  • System goal: Real-time assistive navigation feedback for low-cost wearable deployment
Stack

Python, OpenCV, Mediapipe, Arduino, TFMini LiDAR, BNO085 IMU, BLE/WiFi

Impact

Advances a low-cost wearable navigation aid and demonstrates full-stack capability across hardware, networking, and AI.
